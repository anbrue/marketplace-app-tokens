This app is meant to be a MarketPlace App, extension point Page Builder Context Panel

Development
run `npm install`as usual
run `npm run dev` to run it locally 
You need to have a MarketPlace App connected to localhost (keep in mind the port, default is as usual localhost:3000)

Starter MarketPlace Apps: https://github.com/Sitecore/marketplace-starter
Documentation: https://doc.sitecore.com/mp/en/developers/marketplace/custom-marketplace-apps.html
